# Pineapple Ableton Midi Remote Script
### Setup
```
cd $HOME/Applications/Ableton/Content/App-Resources/Midi-Remote-Scripts
```
```
git clone {{this repo}}
```

### Debug
```
cd $HOME/Library/Preferences/Ableton/{{Live withversion}}
```
```
tail -f Log.txt
```
